=== Milabanners ===
Tags: banners, advertising, ads
Requires at least: 2.0
Tested up to: 2.8.4

== Description ==
 Just a simple plugin to admin your banners.

== Installation ==

Upload the milabbaners plugin to your blog, then activate it from the backend.

 How to use:
 Inside your theme, you can call a banner by calling the next function:
    rotate_banners($group_id);
 THAT'S ALL!
